# Roadmap

This roadmap tracks open-source release readiness and near-term product work. It is not a promise of dates or final scope.

## Completed or In Progress

- Native macOS app shell
- Local managed screenshot library
- Manual image import
- Desktop and Downloads auto import in development builds
- Sidebar collections
- Tags and favorites
- App Trash and Restore
- OCR text extraction
- QR code and link detection
- Search across filenames, OCR text, tags, collections, and detected codes
- PDF export
- Export and share originals
- Duplicate detection and cleanup tools
- Library maintenance and repair tools
- Layout polish for sidebar, grid, toolbar, and inspector
- GitHub release cleanup

## Planned Before v0.1.0-alpha

- Package a downloadable macOS app build
- Review app permissions and onboarding copy
- Add public screenshots to `docs/images/`
- Finish release-focused manual QA
- Reduce or replace remaining development console logging
- Document known limitations and troubleshooting steps
- Confirm signing, notarization, and distribution path
- Add issue templates if needed

## Future Ideas

- More OCR language presets and reprocessing controls
- Smarter organization rules
- More duplicate cleanup views
- Export presets
- Better large-library performance tuning
- Optional sample library for demos
- Future Windows preparation and platform boundary review
